Control de flujo y excepciones

En la programación, pueden ocurrir errores inesperados, como intentar dividir entre cero o acceder a un índice inexistente en un arreglo. Para evitar que el programa se detenga abruptamente, se utilizan bloques de manejo de excepciones como try-catch en Java. Las excepciones permiten capturar y manejar errores de forma controlada, proporcionando mensajes de advertencia o ejecutando código alternativo cuando se detecta un error.

Operadores logicos y relacionales

Los operadores permiten evaluar condiciones dentro de estructuras de control como if, while y for. Se dividen en:

  • Operadores relacionales: Comparan valores y devuelven true o false. Ejemplos: ==, !=, <, >, <=, >=.

  • Operadores lógicos: Permiten combinar condiciones. Ejemplos:
    1. && (and): Devuelve true si ambas condiciones son verdaderas.
    2. || (or): Devuelve true si al menos una condición es verdadera.
    3. ! (not): Invierte el valor de verdad de una expresión.

El uso de operadores lógicos y relacionales es esencial para tomar decisiones dentro del código, permitiendo la creación de flujos de ejecución dinámicos y adaptativos.
